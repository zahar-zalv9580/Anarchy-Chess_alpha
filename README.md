
# ANARCHY CHESS — Alpha Release

> _"Expect the unexpected."_

----------

## Про гру

**Anarchy Chess** — це мультиплеєрна шахова гра, де класичні правила поступово руйнуються під впливом хаосу.

Це не просто шахи.  
Це шахи з **DLC, ефектами, пастками та непередбачуваними подіями**.

----------

## Основні особливості

### Класичні шахи — як основа

-   Повністю робоча логіка ходів
-   Чергування гравців
-   Перемога через знищення всіх **Royal-фігур** (король + ферзь, ~~але скоро буде більше~~) )

----------

### Minesweeper DLC

-   Міни на полі
-   Chain reaction вибухів
-   Цифри-підказки
-   Тактичний ризик кожного ходу
- бімба

----------

### Chess+ DLC

-   Нові ефекти:
    -   Teleport Gun
    -   Bomb Tiles
    -   Fire (поширюється)
    -   Walls (перешкоди)
    -   Pistol
    - куча іншого
-   Повністю змінює стиль гри

----------

### LAN Multiplayer

-   Гра 1v1 по локальній мережі (і по одному WiFi теж! гайд знизу)
-   Host / Join система
-   Серверна логіка

----------

### Візуал і стиль

-   Кастомна кольорова тема (фіолетовий + золотий)
-   Анімації:
    -   рух фігур
    -   вибухи
    -   ефекти
-   Інтро перед запуском гри (фігня тотальна)
- посхалко

----------

### Звук

-   SFX:
    -   ходи
    -   вибухи
    -   взаємодії
    - В І К Н О _ Р О З Б И В А Є Т Ь С Я
-   OST:
    -   меню
    -   гра

----------

## Основна ідея

Кожні кілька ходів у грі активуються **DLC-паки**, які додають нові *механіки*.

`Гравець має адаптуватися до нових правил у реальному часі.`

----------

## Управління

-   ЛКМ — вибір і хід фігури
-   Вибір предметів — під час ходу
-   UI підказує можливі дії

----------

## Умови гри

-   Партія триває: **10 хвилин**
-   Перемога:
    1.  Знищити всі Royal-фігури суперника
    2.  Якщо час вийшов:
        -   більше Royal → перемога
        -   більше матеріалу → перемога
        -   інакше → випадковий результат :D
